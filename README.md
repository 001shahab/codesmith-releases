# Codesmith Releases

Public download site for **Codesmith** installers.

A [3S Holding OÜ](https://3sholding.com) product · Proudly made in Estonia  
Designed & developed by **Prof. Shahab Anbarjafari**

Source code is private. This repository only publishes the compiled apps:

| Platform | File |
|---|---|
| macOS (Apple Silicon) | `Codesmith_*_aarch64.dmg` |
| Windows | `Codesmith_*-setup.exe` |
| Linux (Debian/Ubuntu) | `Codesmith_*_amd64.deb` |

## Download

Open the latest [**Release**](https://github.com/001shahab/codesmith-releases/releases) and pick the installer for your OS.

### First launch (unsigned builds)

- **macOS** — right-click the app → **Open** (Gatekeeper).
- **Windows** — SmartScreen → **More info** → **Run anyway**.
- **Linux** — `sudo dpkg -i Codesmith_*.deb` (or open the `.deb` in your package manager).
